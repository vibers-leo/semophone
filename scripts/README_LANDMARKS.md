# 랜드마크 정보 자동 수집 가이드

## 개요
네이버 플레이스에서 각 매장의 "오시는길" 정보를 자동으로 크롤링하여 2~3줄로 요약하는 스크립트입니다.

## 설치

### 1. Playwright 설치
```bash
npm install -D playwright
npx playwright install
```

### 2. tsx 설치 (TypeScript 실행용)
```bash
npm install -D tsx
```

## 실행 방법

### 전체 매장 크롤링 (40개)
```bash
npx tsx scripts/fetchLandmarks.ts
```

**주의사항:**
- 전체 40개 매장 크롤링 시 약 **2~3분 소요**
- 네이버 차단 방지를 위해 매 요청마다 2초 간격
- 네트워크 상태에 따라 일부 실패 가능

### 출력 결과
1. **콘솔 출력**: 각 매장별 수집 진행상황
2. **JSON 파일**: `scripts/landmarks_output.json` (전체 결과)
3. **TypeScript 코드**: stores.ts에 바로 복붙 가능한 코드

## 수동 대안

크롤링이 실패하거나 일부 매장만 수집하고 싶은 경우:

### 1. 네이버 플레이스 직접 방문
각 매장의 link URL을 브라우저에서 열기

### 2. "오시는길" 정보 복사
페이지에서 위치 안내, 교통 정보 복사

### 3. stores.ts에 수동 추가
```typescript
{
  id: 1,
  name: "...",
  // ...
  landmark: "광명역 2번 출구 150m / 광명CGV 건물 1층",
}
```

## 예시 출력

### 원본 (네이버 플레이스)
```
휴대폰성지 세모폰 서울대입구역점
위치: 서울특별시 관악구 관악로 183
오시는 길: 2호선 서울대입구역 2번 출구에서 도보 3분 거리입니다.
낙성대공원 맞은편에 위치해 있으며, 1층 코너 매장입니다.
```

### 요약 결과
```typescript
landmark: "2호선 서울대입구역 2번 출구 도보 3분 낙성대공원 1층"
```

## 트러블슈팅

### 크롤링 실패 시
1. **네트워크 오류**: 인터넷 연결 확인
2. **차단 오류**: 간격을 3~5초로 늘리기 (코드 수정)
3. **빈 결과**: 네이버 플레이스 페이지 구조 변경 가능성 → 수동 수집

### Playwright 브라우저 설치 오류
```bash
# Chromium만 설치
npx playwright install chromium
```

## 다음 단계

1. **크롤링 실행** → `landmarks_output.json` 생성
2. **결과 확인** → 콘솔에 출력된 TypeScript 코드
3. **stores.ts 업데이트** → 각 매장 객체에 landmark 필드 추가
4. **개발 서버 재시작** → 매장 상세 모달에서 확인
