## 좌표 검증 (보류)

좌표 검증을 실행하려면 네이버 클라우드 콘솔에서 CLIENT_SECRET을 확인하여 `.env.local`에 추가해주세요:

```bash
NAVER_MAP_CLIENT_SECRET=your_client_secret_here
```

그 후 다음 명령어를 실행:

```bash
npx ts-node scripts/verifyCoordinates.ts
```
