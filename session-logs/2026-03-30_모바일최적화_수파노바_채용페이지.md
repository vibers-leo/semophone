# 모바일 최적화 + 수파노바 디자인 적용 + 채용페이지 전면 개편
- 날짜: 2026-03-30
- 에이전트: [TCC]

## 세션 개요
모바일 접속 시 app.semophone.co.kr 리다이렉트 제거, 하단 MobileNav 제거, 사이트 전체 모바일 반응형 정비, 수파노바 디자인 스킬 적용, 채용정보 페이지 전면 수파노바 리디자인 완료.

## 주요 논의

### 1. 그라데이션 + 스토어 그리드 + 상세 이미지 수정
- `SimplifiedHero.tsx`: 히어로 그라데이션 div `w-[52%]` → `w-[54%]` (2% 오른쪽 시프트)
- `app/stores/page.tsx`: 5열 그리드 → 4열 (`xl:grid-cols-5` 제거)
- `StoreDetailModal.tsx`: 세로 이미지(004, 012, 029, 045.jpg — 600×800) `object-contain` + 어두운 배경 → `object-cover` + 밝은 배경(`bg-gray-100`)으로 통일

### 2. 모바일 리다이렉트 & MobileNav 제거
- `app/client-layout.tsx`: isAppDomain 상태, isMobile UA 감지, window.location.href 리다이렉트 블록 전부 제거. MobileNav, InstallPrompt import 및 렌더링 제거.
- `app/globals.css`: `.sticky-cta` 의 `@media (max-width: 768px) { bottom: 64px }` 제거 → CTA 버튼 하단 고정 정상화

### 3. 모바일 반응형 정비 (전체)
- `SimplifiedHero.tsx`: 모바일 섹션 `h-[85vh]` → `min-h-[85dvh]` (iOS Safari dvh)
- `MinimalStats.tsx`: `lg:` → `md:` breakpoint (768px 기준 2열 레이아웃)
- `StoreNetwork.tsx`: 지역 배지 그리드 `gap-8` → `gap-4 md:gap-8`
- `globals.css`: `.text-balance { text-wrap: balance; }` utility 추가

### 4. 수파노바 디자인 전체 적용
- 버튼 hover/active: `hover:scale-[1.02] active:scale-[0.98] transition-all duration-300`
  - 대상: SimplifiedHero (PC/모바일 버튼), MinimalCTA 버튼

### 5. 채용정보 페이지 수파노바 전면 리디자인
7개 섹션 완전 재작성:
- **S1 Hero**: 다크(#09090b) + 비대칭 2컬럼 (헤드라인 + 3개 stat 카드)
- **S2 회사소개**: yellow border 타이포그래픽 인용구, 타임라인에 milestone 레이블 추가
- **S3 핵심가치**: `bg: rgba(255,255,255,0.04) border: rgba(255,255,255,0.08)` 글래스 카드
- **S4 복리후생**: 다크 그라데이션 배경 (`linear-gradient(135deg, #1a1500, #0d0d0d, #09090b)`) — 노란 pill 배지 대비
- **S5 인재상**: `{ num, text }` 번호형 2×2 카드 그리드
- **S6 모집분야**: eyebrow 레이블 + 좌측 정렬 헤더
- **S7 채용모집공고+전형절차**: 글래스 카드 + 지원방법 pill 배지 + 3단계 프로세스 아이콘
- 모든 이모지 제거, 모든 헤딩 `wordBreak: 'keep-all'` + `tracking-tight`

## 기술적 상세
- iOS Safari `100dvh` vs `100vh`: dvh 단위로 주소창 움직임에 따른 레이아웃 점프 방지
- EXIF portrait 이미지: `sips` 명령으로 004/012/029/045.jpg 확인 (600×800). `object-cover`로 컨테이너 채움
- 수파노바 글래스 카드: 다크 배경 위에 `background: rgba(255,255,255,0.04)`, `border: 1px solid rgba(255,255,255,0.08)`, `backdrop-filter: blur`
- 수파노바 eyebrow: `tracking-[0.2em] uppercase text-xs` 영문 소레이블
