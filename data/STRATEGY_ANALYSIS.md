## 1. 현재 상태 진단

### 개발 완성도
- **코드 규모**: 2,731줄 (TypeScript) — 소규모 프로젝트
- **주요 기능**: 40개 매장 지도/검색, 위치 기반 추천, AI 상담(Groq), 구독 관리, PWA 지원
- **기술 스택**: Next.js 16, Firebase, 네이버 Maps API, Groq AI, Resend 이메일
- **상태**: 프로덕션 완성, 도메인 라이브 (semophone.co.kr, app.semophone.co.kr)

### 매출/사용자 현황 (추정)
- **배포 상태**: 완전 라이브, 운영 중
- **사용자**: 월 1000-5000 방문자 추정 (휴대폰 성지 관심층)
- **매출**: 0 KRW (구독 서비스이나 구매 기능 없는 것으로 보임)

### 핵심 과제
1. **수익화 전무** — 구독 서비스라고 명시되어 있으나 결제 로직 없음 (관찰 필요)
2. **니치 시장** — "휴대폰 성지 매장 소개"는 매우 한정된 대상
3. **이용자 확대 어려움** — 일반인은 관심 없고, 타겟(휴대폰 마니아)도 제한적
4. **비즈니스 모델 불명** — "왜 이 서비스가 필요한가"라는 핵심 가치 불명

## 3. Beachhead 분석

**Primary Segment: 휴대폰 마니아 & 개발자 커뮤니티 (특히 중고폰 구매자)**

| 기준 | 평가 | 설명 |
|------|------|------|
| **Burning Pain** | ⚠️ 낮음 | "성지 매장 정보 찾기"가 절실한 문제는 아님. 이미 커뮤니티에서 정보 공유되고 있음 |
| **Willingness to Pay** | ⚠️ 매우 낮음 | 매장 정보는 무료 기대심리. 구독료 내기는 어려움. 반대로 "구매" 링크면 커미션 가능 |
| **Winnability** | ⚠️ 낮음 | 이미 당근마켓, 오늘의집, 네이버 커뮤니티에서 정보 유통 중. 신생 앱은 집중도 낮음 |
| **Referral Potential** | ⚠️ 낮음 | "좋은 폰 샀다" 공유는 있으나, 앱 추천까지는 약함 |

**2순위: 해외 여행객**
- 한국 휴대폰 상황을 궁금해하는 외국인
- 하지만 앱이 한글 기반이므로 영문화 필요
- 매우 작은 세그먼트

## 4. Growth Loop

**Community-Driven + Word-of-Mouth**

```
휴대폰 마니아가 세모폰 발견
  ↓
위치 기반으로 가까운 매장 3곳 확인
  ↓
"여기서 싼데?" AI 상담 활용
  ↓
실제 매장 방문 → 구매
  ↓
커뮤니티(당근마켓, 오늘의집)에서 "세모폰으로 찾았어" 공유
  ↓
다른 마니아가 앱 설치
```

**문제**: 루프가 약함. "매장 정보"만으로는 반복 사용 이유 없음 (매달 폰을 새로 안 사니까).

**해결책**: (1) **AI 상담**을 매달 사용하는 "휴대폰 뉴스레터"로 전환, (2) **커뮤니티** 기능 추가 (사용자 리뷰, 매장 평점)

## 5. 가격 전략

### 현재 상태
- **표시**: "구독 서비스" 명시
- **실제**: 결제 기능 없는 것으로 보임 (구현 미완료?)

### 권장 가격 모델

**Option 1: 프리미엄 타이어 (추천)**
| 플랜 | 가격 | 기능 |
|------|------|------|
| **Free** | ₩0 | 매장 검색, AI 상담 1회/일 |
| **Pro** | ₩4,900/월 | AI 상담 무제한, 찜하기, 매장 리뷰 작성 |

**Option 2: 광고 모델 (장기)**
- Free 사용자: 매장 프로필에 "일반 노출"
- 매장주 유료화: 매장 프리미엄 광고 (월 ₩500k) — B2B 수익

### 현실적 조언
- 현재 니치 시장이므로 수익화는 **2년 후**에나 가능
- 지금은 **사용자 기반 확대** → **광고 모델** 전환이 현실적
- 구독 모델로는 월 활성 사용자 1000명 필요한데, 현재 추정 100-500명 수준

## 7. North Star Metric

### 권장 NSM
**월간 활성 사용자당 평균 매장 검색 횟수 (Monthly Searches per MAU)**

- 정의: 월간 1회 이상 활동한 사용자가 평균 몇 번 매장을 검색했는가
- 목표: 6개월 내 5회/사용자/월 달성 (주 1회 이상 활동)
- 이유: (1) 사용자 관심도 증명, (2) 반복 사용 강도, (3) 광고 모델 확장 가능성

### Input Metrics (주간 추적)
1. **월간 활성 사용자 (MAU)**: 월 500명 이상 (초기 목표)
2. **매장 검색 완료율**: 매장 접근 시 상세 정보 확인 비율 30% 이상
3. **AI 상담 사용률**: 활성 사용자 중 50% 이상 월 1회 이상 사용
4. **리테인션 (D30)**: 신규 사용자 중 30% 이상 30일 후 재방문
5. **커뮤니티 공유**: 월간 SNS/커뮤니티 언급 5회 이상

## 8. 3단계 로드맵

### Phase 1: 기초 검증 & 커뮤니티 확대 (현재~5월, 2개월)

**목표**: MAU 500명 달성, 커뮤니티 기반 구축

**실행 항목**
- [ ] 현재 사용자 10명 인터뷰 (왜 쓰나? 불편한 점은? 결제할 의사는?)
- [ ] 당근마켓/오늘의집/클리앙 "휴대폰" 커뮤니티에서 세모폰 소개 (스팸 아닌 진정성 있게)
- [ ] 매장 정보 업데이트 프로세스 자동화 (크라우드소싱 또는 API)
- [ ] AI 상담 품질 개선 (사용자 피드백 기반)
- [ ] 앱 설치 추적 (Google Play 앱스토어 없다면 웹 PWA 홍보)

**성공 지표**
- MAU 500명 달성
- AI 상담 완료율 40% 이상
- 커뮤니티 언급 월 10회 이상

### Phase 2: 수익화 파일럿 & 매장 B2B (5월~8월, 3개월)

**목표**: 첫 매장 광고 구매, 월 ₩1m 수익 초기

**실행 항목**
- [ ] 매장주 대상 "프리미엄 노출" 광고 패키지 디자인 (월 ₩500k)
- [ ] 상위 10개 매장에 직접 영업 (전화, 카톡)
- [ ] 사용자 구독 모델 소프트 런칭 (Pro ₩4,900/월, AI 무제한)
- [ ] 리뷰/평점 시스템 추가 (매장 신뢰도 증가)
- [ ] 배포 경로 다양화 (웹 + App Store 검토)

**성공 지표**
- 매장 광고 구매 2곳 이상 (월 ₩1m 초기 달성)
- 사용자 유료 전환 10명 이상
- MAU 1000명 달성

### Phase 3: 마켓플레이스 진화 (8월~2027년 1월, 6개월)

**목표**: 월 ₩5m+ 수익 (매장 광고 + 사용자 구독), 매장 수 100개 이상

**실행 항목**
- [ ] "매장에서 폰 팔기" 마켓플레이스 추가 (커미션 모델)
- [ ] 매장별 상품 연동 (인벤토리)
- [ ] 매장 구매 유도 버튼 추가 ("이곳에서 구매" → 쿠팡/당근마켓 링크)
- [ ] 해외 확장 검토 (English version)
- [ ] 웹/앱 네이티브 앱 출시 (React Native)

**성공 지표**
- 월 매출 ₩5m+ (광고 + 구독 + 마켓플레이스)
- 매장 수 100개 이상
- MAU 5000명 이상

## 핵심 한 줄 요약

> **"휴대폰 성지를 한눈에 찾고, AI와 대화하며 선택하는 한국식 폰 쇼핑 가이드"**

### 현재 위험도: 🔴 높음
- **강점**: 기술 완성도 높음, PWA 준비됨, 흥미로운 AI 상담 기능
- **약점**: 비즈니스 모델 불명, 타겟 고객 검증 미완료, 수익 경로 부재
- **기회**: "폰 마니아 커뮤니티" 기반으로 매장 마켓플레이스로 성장 가능

### 투자 결정 기준
1. **양수 신호**: Phase 1에서 MAU 500명 + 사용자 인터뷰 "유용해" 피드백 → Phase 2 진행
2. **음수 신호**: 커뮤니티 아웃리치 후 관심 <5% + 인터뷰 결과 "굳이 필요 없어" → 방향 재검토
3. **재검토 신호**: "수익화"가 미션이면 지금 바로 매장 B2B 영업 시작해야 함 (Time to Value 중요)

**최종 조언**:

세모폰은 **가장 위험한 프로젝트**입니다. 기술은 완성됐지만 **사업 검증이 전무**합니다. 지금 바로 할 일:

1. **고객 인터뷰 10명** (당근마켓, 오늘의집에서 모집)
   - "이 앱 왜 쓰세요?"
   - "폰을 살 때마다 성지를 검색하나요?"
   - "월 ₩5,000 내고 쓰실래요?"

2. **매장주 인터뷰 5명**
   - "광고비 월 ₩500k 쓸 의사 있나요?"
   - "어떤 정보가 필요한가요?"

3. **커뮤니티 아웃리치** (당근마켓/오늘의집)
   - 스팸이 아닌 진정성 있는 소개

이 3가지를 2주 내에 완료하면 이 서비스가 정말 필요한지, 아니면 "공중에 떠있는 프로젝트"인지 명확해집니다. 만약 인터뷰 결과가 양수면 매장 B2B 영업을 즉시 시작하세요. (광고/마켓플레이스는 사용자가 아닌 **매장주**를 만족시키는 것이 먼저입니다.)
